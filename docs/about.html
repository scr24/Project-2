About The Project

This assignment is for IS117-101 Fall 2020. This project is designed to teach us how our information is collected on the web. While some of the times that can benefit us, in some cases it can be harmful. We will cover some cases where that data is used and how you can choose to remain anonymous if you don't see benefit.
